# 🏎️ F1 News Bot

Telegram-бот, который автоматически мониторит новостные сайты о Формуле 1, оценивает «хайп» каждой новости через ChatGPT и присылает самые горячие в Telegram. Генерирует готовые посты для канала одной кнопкой.

## Возможности

- 📡 **Автоматический мониторинг** — парсинг RSS-лент 8 топовых F1-сайтов каждые N минут
- 🤖 **AI-анализ хайпа** — ChatGPT оценивает каждую новость по шкале 1-10
- 🔥 **Мгновенные алерты** — новости с хайпом ≥ 8 сразу приходят в Telegram
- 📋 **Дайджест** — команда `/digest` покажет новости с хайпом 3-7 за текущий день
- ✍️ **Генерация постов** — одной кнопкой ChatGPT создаёт пост на русском для канала
- ✏️ **Редактирование** — можно подправить текст перед публикацией
- 🖼 **Фото** — можно прикрепить картинку к посту
- 📤 **Публикация в канал** — отправка готового поста одной кнопкой
- 🎯 **Контекст канала** — при генерации ChatGPT видит последние 7 постов (сохраняет стиль, не повторяется)
- ↩️ **Reply на связанные посты** — ChatGPT автоматически находит связанный пост + ручной выбор кнопкой (пагинация по 5)
- 🔒 **Приватность** — все команды и кнопки доступны только владельцу бота
- 🧠 **Двухмодельный AI** — gpt-5-mini для анализа, gpt-5.2 для генерации постов
- ⚡ **Prompt caching** — оптимизация промптов для автоматического кеширования OpenAI (экономия ~30-50% на input-токенах)

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/RamanovskiyVV/f1-news-bot.git
cd f1-news-bot
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/Mac
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

```bash
copy .env.example .env       # Windows
# cp .env.example .env       # Linux/Mac
```

Заполните `.env`:

| Переменная | Описание | По умолчанию |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) | — |
| `TELEGRAM_CHANNEL_ID` | ID канала (`@username` или `-100xxx`) | — |
| `OPENAI_API_KEY` | API-ключ OpenAI | — |
| `OPENAI_MODEL` | Модель OpenAI (анализ, поиск связей) | `gpt-5-mini` |
| `OPENAI_MODEL_GENERATE` | Модель OpenAI (генерация постов) | `gpt-5.2` |
| `CHECK_INTERVAL_MINUTES` | Интервал проверки (мин) | `10` |
| `HYPE_THRESHOLD` | Минимальный порог хайпа для алертов | `8` |

> **Важно:** Бот должен быть администратором канала для публикации постов.

### 5. Запуск

```bash
python main.py
```

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и справка |
| `/check` | Проверить новости прямо сейчас |
| `/digest` | Показать новости с хайпом 3-7 за сегодня |
| `/clear` | Скрыть просмотренные дайджест-новости |
| `/status` | Статус бота |

## Как работает

```
RSS-ленты 8 F1-сайтов
        ↓
   Сбор новостей (feedparser + httpx)
        ↓
   Дедупликация (seen_news.json)
        ↓
   Анализ через ChatGPT API
   (оценка хайпа 1-10 + саммари на русском)
        ↓
   ┌─────────────────────────────────┐
   │  Хайп ≥ 8 → алерт в Telegram   │
   │  Хайп 3-7 → дневной кэш        │
   │             (доступен по /digest)│
   │  Хайп 1-2 → пропускается       │
   └─────────────────────────────────┘
        ↓
   [🔥 Хайп: 9/10]
   [Краткое описание на русском]
   [🔗 Ссылка на оригинал]
   [Кнопка: ✍️ Сгенерировать новость]
        ↓
   ChatGPT генерирует пост для канала
   (с учётом последних 7 постов канала)
        ↓
   [📤 Отправить в канал]
   [✏️ Редактировать] [🔄 Перегенерировать]
   [🖼 Картинка] [↩️ Reply]
        ↓
   При публикации: ChatGPT ищет связанный пост
   → подтверждение → reply в канале
```

## Шкала хайпа

| Оценка | Уровень | Примеры |
|---|---|---|
| 10 | 🔥🔥🔥 Сенсация | Смена пилота топ-команды, серьёзная авария, дисквалификация |
| 8-9 | 🔥🔥 Очень важно (алерт) | Победа в гонке, поул, контрактные новости, тех. инновации |
| 5-7 | 🔥 Интересно (дайджест) | Предквалификационные расклады, обновления болидов |
| 3-4 | 📰 Обычное (дайджест) | Пресс-конференции, рутинные обновления |
| 1-2 | Малозначительное | Промо, спонсорские новости |

## Стоимость API

Двухмодельный подход: `gpt-5-mini` (анализ) + `gpt-5.2` (генерация):

| | Модель | Расчёт | В месяц |
|---|---|---|---|
| Анализ новостей | gpt-5-mini | ~8 батчей/день × 30 дней | ~$0.52 |
| Генерация постов | gpt-5.2 | ~3-5 постов/день | ~$0.87 |
| Поиск связанных постов | gpt-5-mini | ~3-5 публикаций/день | ~$0.05 |
| **Итого** | | | **~$1.00 – $1.50** |

> Prompt caching экономит ~30-50% на input-токенах. При очень активном использовании — до $2-3/мес.
